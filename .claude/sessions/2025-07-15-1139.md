## Tips for Future Developers

### 🏪 **Working with Stores**
```typescript
// Import stores from central location
import { useUIStore, useVideoWizardStore } from '@/lib/stores'

// Use selective subscriptions for performance
const theme = useUIStore(state => state.theme)

// Access store actions directly
const { addNotification } = useUIStore()
```

### 🔄 **Store Subscriptions**
```typescript
// Use the master subscription hook in app layout
import { useStoreSubscriptions } from '@/lib/stores/subscriptions'

// This sets up all cross-store synchronization
useStoreSubscriptions()
```

### 🛠️ **Development Tools**
- Redux DevTools available in browser for store debugging
- `getStoreSnapshot()` function for programmatic state inspection
- `resetAllStores()` for development testing (dev mode only)
- `logAllStores()` for console debugging

### 📝 **Best Practices**
- Always use typed selectors for performance
- Prefer domain-specific hooks over direct store access
- Use persistence partializing to avoid storing UI state
- Implement optimistic updates for better UX
